Enum for operations on BitRegister.

Valid values are as follows:

AND = <RegWiseOp.AND: '&'>#

EQ = <RegWiseOp.EQ: '=='>#

LT = <RegWiseOp.LT: '<'>#

SUB = <RegWiseOp.SUB: '-'>#
MUL = <RegWiseOp.MUL: '*'>#
DIV = <RegWiseOp.DIV: '/'>#
POW = <RegWiseOp.POW: '**'>#
LSH = <RegWiseOp.LSH: '<<'>#
RSH = <RegWiseOp.RSH: '>>'>#
NOT = <RegWiseOp.NOT: '~'>#
