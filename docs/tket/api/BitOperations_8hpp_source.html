28 // Application: we have a mask like 000101100111001.
29 // Those positions with "1" are "forbidden";
30 // the other positions are "free": we are allowed to set them.
31 // In the example 000101100111001, there are 7 forbidden positions
32 // (ones) and 8 free positions (zeros).
33 // If we have a length 8 binary string "abcdefgh",
34 // we want to insert zeros and stretch it to become abc0d00ef000gh0
35 // (so that the forbidden positions are all zero and thus may be set
36 // by some other function by a simple OR).
37 //
38 // This is done by ANDing abcdefgh with 00000011, 00001100, 00010000,
39 // 11100000 in turn, shifting each result left by a certain number
40 // of bits, then ORing them together.
41 // In each element, "first" is the mask that we must AND with;
42 // "second" is the argument to the shift left operator.
43 typedef std::vector<std::pair<SimUInt, unsigned>> ExpansionData;
