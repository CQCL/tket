38// 1) representing end-of-circuit measurements as a collection of single Z
39// operators 2) transforming each fo these single Z operators to a new Pauli
40// operator by acting on them with an end-of-circuit Clifford subcircuit 3)
41// finding a Clifford that mutually diagonalises these Pauli operators 4)
42// replacing the end-of-circuit Clifford subcircuit with the mutual
43// diagonalisation circuit and adding classical logic to map the diagonalised
44// Pauli operators back to the desired single Z operators
