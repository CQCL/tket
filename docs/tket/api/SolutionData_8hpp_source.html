89  // If upper/lower bounds for the scalar product are equal,
90  // then it's effectively the standard pure subgraph monomorphism problem;
91  // there's no point in evaluating scalar products for different solutions
92  // because they're all equal,
93  // and no point using a weight nogood detector.
94  //
95  // But if we allow zero weights, then this is NOT quite equivalent to
96  // "the pattern weights are constant, and the target weights are constant".
97  // e.g. consider p-weights = [0,1,1] and t-weights = [1,1,1].
98  // Even though the p-weights are NOT constant,
99  // any valid assignment will give scalar product 2 exactly,
100  // and (in a simple case like this) it will calculate L=U=2.
