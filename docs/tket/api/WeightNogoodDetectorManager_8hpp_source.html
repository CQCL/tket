119  // The below values of the parameters are little more than

124  // "pk" numbers are "per K", where K=1024,
125  // rather than "per 100" (ordinary percentages).
126  // This is preferable to a power of ten because
127  // (1) it can give faster int division

129  // (2) allows lossless calculations in most cases.
