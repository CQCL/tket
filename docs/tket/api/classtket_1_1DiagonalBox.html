Construct a circuit that synthesise the given unitary diagonal operator.

Parameters
diagonalthe diagonal entries of the operator
upper_trianglethe diagonal operator will be decomposed as a sequence of multiplexed-Rz gates. This argument decides whether the multiplexed-Rz gates take the shape of an upper triangle or a lower triangle.
