For some Circuit, returns maps between Circuit UnitID and Architecture UnitID that can be used for reassigning UnitID in Circuit.

Maps are constructed by running a Weighted Subgraph Monomorphism for the given problem and returning up to matches number of potential solutions, ranked. Additionally, the top x mappings with identical WSM score is costed depending on passed Device characteristics, effecting the ranking.

Parameters
