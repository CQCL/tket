Detailed Description

Reset operation defined by a pair of anti-commuting strings For example, a tket Reset OpType can be defined as a Z/X pair.

The Pauli Z can be seen as a Z-basis measurement, and the Pauli X can be seen as the post measurement conditional X gate.
