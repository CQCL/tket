tket::UnitaryTableau::UnitaryTableau ( unsigned  n)
explicit

An implementation of the stabilizer-destabilizer tableau for unitary Cliffords in Aaronson & Gottesman, "Improved Simulation of Stabilizer Circuits", https://arxiv.org/pdf/quant-ph/0406196.pdf.

If a Pauli is placed at an input before the unitary, there is a Pauli string over the outputs which would have an equivalent effect. The rows correspond to these Pauli strings for X and Z on each input qubit. In other terms, the ith X row is the (phaseful) Pauli string P such that P C X_i = C, and similarly for the Z rows.

The Z rows generate the stabilizer group for the state prepared when the unitary is applied to the initial |0>^n state, with the X rows extending this to generate the full n-fold Pauli group.

Qubits indexed using Qubit objects. Construct the tableau for the identity over n qubits (given default qubit names).
