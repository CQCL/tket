Detailed Description

Encapsulation of a unitary stabilizer tableau as a Box to use within a circuit.

Implementation and decomposition currently use the methods from Aaronson & Gottesman, "Improved Simulation of Stabilizer Circuits", https://arxiv.org/pdf/quant-ph/0406196.pdf which yield high gate counts in practice.
