Returns information about the degrees of vertices up to the given distance from the root vertex.

If this is a pattern graph, returns data for vertices EXACTLY at the specified distance; if a target graph, for vertices at or closer than the given distance. Requires the distance to be >= 2.

const std::vector< VertexWSM > & tket::WeightedSubgraphMonomorphism::NearNeighboursData::get_vertices_at_distance ( VertexWSM  v,

Calculated lazily, on demand; returns a sorted list of vertices exactly at distance d from v.

We must have d >= 2 (neighbours with d=1 come from the NeighboursData object instead).

Parameters
vA vertex in the graph.
distanceA distance d >= 2.
Returns
A reference to a sorted vector (stored within this class) of all vertices at distance exactly d from v. The reference may be invalidated, of course, if further non-const calls are made to this class.
