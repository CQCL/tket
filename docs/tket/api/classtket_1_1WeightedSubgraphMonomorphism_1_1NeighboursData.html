Detailed Description

The main object used to search for neighbours of a vertex, check for existing edges, and get edge weights.

The graph is undirected. No loops or multiple edges are allowed. The input vertices MUST be {0,1,2,...,N} for some N. Will throw if not. There cannot be any isolated vertices (more precisely, since edges are passed in and vertices are deduced from them, it's simply not possible for this class to know anything about isolated vertices).

◆ NeighboursData()

tket::WeightedSubgraphMonomorphism::NeighboursData::NeighboursData ( const GraphEdgeWeights &  edges_and_weights)
explicit

Initialise with a given graph with edge weights.

The graph is undirected, therefore edges (v1,v2) and (v2,v1) are counted as the same. It's not necessary to pass in both pairs, but if both ARE passed in, then the weights must be equal, and this is checked. Note that edge weights are nonnegative, but also allowed to be zero. An edge weight 0 has no special meaning.

Parameters
edges_and_weightsAn edge->weight map ("edge" just being a pair of VertexWSM).
