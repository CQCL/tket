Detailed Description

template<class T>
class tket::WeightedSubgraphMonomorphism::ReusableStorage< T >

For storing objects of type T, which can be reused.

Objects are "released", i.e. marked as ready for reuse - like garbage collection - rather than being erased, but the caller is responsible for clearing such T objects when making use of them again.

Access the objects by a size_t ID, in time O(1). The IDs are allowed to be reused, and references are allowed to be invalidated if other elements are added or "erased" (released). An ID remains valid (unlike a reference) even as others are added or released, until the object with that ID is released.

So, this is useful if we frequently need objects like std::vector which are much cheaper to clear and reuse than to construct afresh.
