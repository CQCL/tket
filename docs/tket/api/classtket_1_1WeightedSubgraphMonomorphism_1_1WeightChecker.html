Detailed Description

This wraps both a WeightNogoodDetector object (used to estimate weights, and hence break off early if we can prove that any solution from this point would be rejected, for having too high a scalar product), and a WeightNogoodDetectorManager object (used to monitor progress, and decide whether or not we even want to attempt detection).
