Detailed Description

Data for an undirected graph.

Once stored inside here, the data has automatically been checked and cleaned: the vertices in neighbour lists actually exist, there are no missing vertices, no duplicate edge data, etc. The vertices are {0,1,2,...,v-1}. The number of vertices must be known at the start (or, it can be reset later, but only by clearing all data), so it is not completely dynamic. The constructors throw upon invalid data.
