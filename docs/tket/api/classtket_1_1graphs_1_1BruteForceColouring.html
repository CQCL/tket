Detailed Description

For colouring vertices in a single connected component of a graph.

Although it's brute force, because it goes down the vertices in a fixed order, as specified by "ColouringPriority", (like traversing a "game tree"), there is the potential for pruning, and hence it can be much quicker than simply trying every possible colouring. It only looks for ONE colouring.
