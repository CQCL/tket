Detailed Description

template<typename T>
class tket::graphs::DirectedGraph< T >

DirectedGraph instances are loop-free directed graphs.

It is a wrapper around a BGL graph that provides a clean class API, taking care of mapping all BGL vertices and edge pointers to nodes, respectively pairs of nodes.

The vertices and edges can be given weights of type double if desired, and the underlying undirected graph can be computed.

All functionality for this class is implemented in the base class DirectedGraphBase. This class only adds caching of some function calls for efficiency, invalidating cache in case of changes on the underlying graph.
