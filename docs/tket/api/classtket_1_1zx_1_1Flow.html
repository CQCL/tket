Detailed Description

Data structure for flow in qubit MBQC.

Different classes of flow exist based on the types of measurements and correction sets accepted, but the contents of the flow are the same. Causal < XY gflow < 3Plane gflow < Pauli flow

c defines the correction set for each measured vertex. d approximates the partial order by giving the depth of the measurement from the output, i.e. d(u) < d(v) => v is measured before u.
