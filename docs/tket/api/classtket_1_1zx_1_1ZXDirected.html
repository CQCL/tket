Detailed Description

Virtual subclass of ZXGen for directed (non-commutative) generators.

The generator has a pre-determined number of ports labelled from 0 to n_ports-1. Each port has a pre-determined QuantumType, captured by the signature. There must be exactly one incident edge for each port and it must match the corresponding QuantumType. Implementations include both actual generators (e.g. Triangle) and Box types
