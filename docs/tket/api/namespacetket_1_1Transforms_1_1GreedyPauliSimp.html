Converts the given circuit into a GPGraph and conjugates each node by greedily applying 2-qubit Clifford gates until the node can be realised as a single-qubit gate, a measurement, or a reset.

The final Clifford operator is synthesized in a similar fashion. Allows early termination from a thread via a stop_flag.
