typedef std::forward_list<NeighboursAndCounts> tket::WeightedSubgraphMonomorphism::DerivedGraphStructs::NeighboursAndCountsStorage

We create derived graphs lazily (since many vertices and edges might never be needed).

For convenience, we want to pass around references or pointers, but we can't just store the objects in a vector because the references might be invalidated upon insertions. Thus we use a linked list.
