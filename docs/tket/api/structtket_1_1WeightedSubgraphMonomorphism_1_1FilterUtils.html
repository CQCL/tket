Given sorted degree sequences (i.e., the vertex degrees of the neighbours) of vertices P_v and T_v, is it maybe possible to map P_v to T_v, based upon the sequences?

Parameters
pattern_v_deg_seqThe increasing vertex degrees of the neighbours of PV
target_v_deg_seqThe increasing vertex degrees of the neighbours of TV
Returns
True if the target sequence dominates the pattern sequence, i.e. in the absence of other graph theoretic information, there is no obstruction to mapping PV->TV.
