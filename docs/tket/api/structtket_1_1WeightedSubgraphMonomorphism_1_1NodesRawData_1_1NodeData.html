◆ unassigned_vertices_superset

std::vector<VertexWSM> tket::WeightedSubgraphMonomorphism::NodesRawData::NodeData::unassigned_vertices_superset

For performance: rather than searching through EVERY Dom(PV), which will include many assigned PV (for which Dom(PV)={y}), we have a list of all those PV known to have |Dom(PV)| > 1.

However, we don't need a strictly accurate list. This is EITHER empty, OR guarantees that every PV with size Dom(PV) > 1 is included in here. However, some extra PV with |Dom(PV)| = 1 may also be included.

Note that this only works because, as we move down the search tree, the domain sets reduce (so we can restrict to a previous list of PVs).
