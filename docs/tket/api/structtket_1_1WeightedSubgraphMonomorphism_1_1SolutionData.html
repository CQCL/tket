◆ finished

bool tket::WeightedSubgraphMonomorphism::SolutionData::finished = false

If true, the search is over; EITHER we've found a (joint) OPTIMAL solution, OR we've proved that there is NO solution.

If false, then our solution (if any) is merely the best found so far, not necessarily optimal. But, if the "terminate_with_first_full_solution" solver option was chosen and a solution was found, so that it terminated early, this will still be set to "false", so that the caller can continue searching for more solutions if desired.
