Detailed Description

If we have vertices {0,1,2,...,N} instead of arbitrary integers, then we can use std::vector instead of std::map in many places.

This is actually a significant speedup (>10%). Thus, when we pass in a problem, we relabel internally to ensure that this holds; and convert back to the original vertices when the user requests a solution. This just contains data to do the conversions.
