49 * remove that qubit and continue until the tableau contains no qubits left.
50 * This gradually builds up a set of operations both before and after the
51 * working tableau. We should ask for the following combination of temporary
52 * states to compose to form a channel equivalent to that described by the
53 * input tableau:
54 * - in_circ: a unitary circuit (without implicit permutations)
55 * - post_selected: a set of post-selection actions to apply to the outputs of
56 * in_circ
57 * - discarded: a set of discard actions to apply to the outputs of in_circ
58 * - collapsed: a set of collapse actions to apply to the outputs of in_circ
59 * (i.e. decoherence in the Z basis)
60 * - tab: the remaining tableau still to be solved. Acting as an identity on
61 * any qubits not contained within tab
62 * - in_out_permutation: a permutation of the qubits, read as a map from the
63 * input qubit name to the output qubit it is sent to
64 * - zero_initialised: a set of initialisations of fresh output qubits
65 * (in_out_permutation may join these qubits onto input qubits that have been
66 * post-selected or discarded to reuse qubits)
67 * - mix_initialised: a set of initialisations of output qubits into
68 * maximally-mixed states (in_out_permutation may similarly join these onto
69 * input qubits no longer in use)
