25 // Something like this is needed for proper random test data generation
26 // if you want to be platform-independent, as the C++ standard is stupid.
27 // (A major scandal, in my opinion).
28 // The random engines are mostly guaranteed by the standard,
29 // but the DISTRIBUTIONS, e.g. uniform_distribution, are NOT
30 // (i.e., the actual algorithm used to convert a string of bits to a number
31 // in the range {0,1,2,...,N} is not specified at all by the C++ standard).
32 // Thus, we are NOT guaranteed to get the same results, even with the same
33 // (1) engine; (2) initial seed; (3) distribution,
34 // if we use different platforms (or even different compilers
35 // on the SAME platform), or even different compiler VERSIONS!!!
36 //
37 // The C++ standard as far as I know does not specify ANY distribution
38 // implementations, not even optionally, so you HAVE to do this yourself,
39 // even for something as simple as a uniform distribution.
40 // The same applies to, e.g., std::random_shuffle.
